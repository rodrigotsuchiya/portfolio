jokenpo

Descriçao do projeto

este projeto foi desenvolvido no curso online programar do zero do devSamurai, onde desenvolvias minhas habilidades adquiridas durante o curso.

neste projeto foi desenvolvido o famoso jogo Jokenpo(pedra, papel, tesoura), o jogo consiste em dois jogadores, no nosso caso o jogador e o computador, onde eles escohem entre pedra, papel ou tesoura, para sabre quem venceu:

  • pedra quebra tesoura
  • tesoura corta o papel
  • papel embulha a pedra
